# ShowDex Movie Database (MDB)

Welcome to the data repository for **ShowDex**. This repository contains the complete, curated movie metadata used to power the ShowDex application. 

The dataset is provided entirely for free and is open to public use, distribution, and contribution.

---

## Data Sources & Curation

The movie information available in this dataset has been aggregated, cleansed, and verified across multiple channels:

1. **First-hand viewing & curation** – Direct insights and structural entry from manual tracking.
2. **Wikipedia** – Baseline public information registries and release histories.
3. **Public web directories** – Broad online catalogs used to cross-reference and populate deeper attributes.

## Data Quality & Contributions

Every effort has been made to ensure that the dataset is clean, well-formatted, and accurate. However, data anomalies or typos can still occur. 

If you notice any entries that require cleansing, formatting corrections, or factual updates:
* Please Open an issue describing the incorrect data.
* Pull requests are highly welcome if you would like to submit formatting fixes or missing metadata directly.

---

## Data Scope & Privacy Policy

> ⚠️ **Important Privacy Notice:** 
> This repository contains **only general movie metadata** (titles, release years, credits, and descriptors). No user-specific data—such as user personal profiles, watchlists, account information, or personal application ratings—is stored, tracked, or shared here. All user personal data remains strictly confidential and private.
